# Step 04 — Base Layout & Shell Components

Build the shared layout shell: `BaseLayout.astro`, `TopNavBar.astro`, `LeftSidebar.astro`, and `Footer.astro`. Every page uses these.

**Prerequisite:** Steps 01–03 complete.

---

## 4.1 `src/layouts/BaseLayout.astro`

The outer shell for every page. Accepts props for `<title>` and active nav state.

```astro
---
import '../styles/global.css';
import TopNavBar from '@components/layout/TopNavBar.astro';
import LeftSidebar from '@components/layout/LeftSidebar.astro';
import Footer from '@components/layout/Footer.astro';
import profile from '../content/data/profile.json';

interface Props {
  title: string;
  activeNav: 'home' | 'experience' | 'projects' | 'blog' | 'contact';
  description?: string;
}

const {
  title,
  activeNav,
  description = 'Personal portfolio of Ondřej Burda — software developer.',
} = Astro.props;

const siteUrl = 'https://burdis28.github.io';
const canonicalUrl = new URL(Astro.url.pathname, siteUrl).href;
---

<!DOCTYPE html>
<html lang="cs" class="light">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>{title}</title>

  <!-- SEO -->
  <meta name="description" content={description} />
  <link rel="canonical" href={canonicalUrl} />

  <!-- Open Graph -->
  <meta property="og:type" content="website" />
  <meta property="og:url" content={canonicalUrl} />
  <meta property="og:title" content={title} />
  <meta property="og:description" content={description} />
  <meta property="og:image" content={`${siteUrl}/images/og-card.jpg`} />

  <!-- Favicon -->
  <link rel="icon" type="image/svg+xml" href="/favicon.svg" />

  <!-- Sitemap (generated by @astrojs/sitemap) -->
  <link rel="sitemap" href="/sitemap-index.xml" />
</head>

<body class="bg-surface font-body text-on-surface min-h-screen flex flex-col">
  <TopNavBar activeNav={activeNav} />

  <main class="flex-1 max-w-7xl mx-auto w-full px-6 pt-32 pb-12">
    <div class="grid grid-cols-1 lg:grid-cols-12 gap-12">

      <!-- Left sidebar — fixed on desktop, stacked on mobile -->
      <aside class="lg:col-span-3">
        <div class="lg:sticky lg:top-24">
          <LeftSidebar />
        </div>
      </aside>

      <!-- Main content area -->
      <div class="lg:col-span-9">
        <slot />
      </div>

    </div>
  </main>

  <Footer />
</body>
</html>
```

Notes:
- `lang="cs"` — set to Czech (change to `"en"` if content is in English)
- `lg:sticky lg:top-24` on sidebar wrapper — sidebar stays visible while scrolling on desktop
- `<slot />` — Astro's content projection; page content renders here

## 4.2 `src/components/layout/TopNavBar.astro`

Fixed frosted-glass navigation bar. Highlights the active route.

```astro
---
import profile from '../../content/data/profile.json';

interface Props {
  activeNav: 'home' | 'experience' | 'projects' | 'blog' | 'contact';
}
const { activeNav } = Astro.props;

const navLinks = [
  { id: 'home',       label: 'Home',       href: '/' },
  { id: 'experience', label: 'Experience', href: '/experience' },
  { id: 'projects',   label: 'Projects',   href: '/projects' },
  { id: 'blog',       label: 'Blog',       href: '/blog' },
  { id: 'contact',    label: 'Contact',    href: '/contact' },
];
---

<nav class="fixed top-0 w-full z-50 bg-white/80 backdrop-blur-md border-b border-outline-variant/15 shadow-sm">
  <div class="max-w-7xl mx-auto px-6 h-16 flex items-center justify-between">

    <!-- Brand wordmark -->
    <a href="/" class="text-xl font-black text-primary font-headline tracking-tight hover:opacity-80 transition-opacity">
      Ondřej Burda
    </a>

    <!-- Desktop nav links -->
    <div class="hidden md:flex items-center space-x-8">
      {navLinks.map((link) => (
        <a
          href={link.href}
          class={
            activeNav === link.id
              ? 'text-primary border-b-2 border-primary pb-1 font-label text-sm font-bold'
              : 'text-on-surface-variant hover:text-primary transition-colors font-label text-sm font-medium'
          }
        >
          {link.label}
        </a>
      ))}
    </div>

    <!-- Right actions -->
    <div class="flex items-center gap-3">
      <!-- Mobile hamburger (visible below md) -->
      <button
        id="mobile-menu-btn"
        class="md:hidden p-2 rounded-lg hover:bg-surface-container-low transition-colors text-on-surface-variant"
        aria-label="Toggle navigation menu"
        aria-expanded="false"
        aria-controls="mobile-menu"
      >
        <span class="material-symbols-outlined">menu</span>
      </button>

      <!-- Desktop: avatar -->
      <div class="hidden md:block w-8 h-8 rounded-full overflow-hidden border border-outline-variant/30">
        <img
          src={profile.avatarUrl}
          alt={profile.name}
          class="w-full h-full object-cover"
          width="32"
          height="32"
        />
      </div>
    </div>

  </div>

  <!-- Mobile dropdown menu -->
  <div
    id="mobile-menu"
    class="hidden md:hidden border-t border-outline-variant/15 bg-white/95 backdrop-blur-md"
  >
    <div class="max-w-7xl mx-auto px-6 py-4 flex flex-col gap-1">
      {navLinks.map((link) => (
        <a
          href={link.href}
          class={
            activeNav === link.id
              ? 'px-4 py-3 rounded-lg bg-primary/5 text-primary font-label text-sm font-bold'
              : 'px-4 py-3 rounded-lg text-on-surface-variant hover:bg-surface-container-low hover:text-primary transition-colors font-label text-sm font-medium'
          }
        >
          {link.label}
        </a>
      ))}
    </div>
  </div>
</nav>

<!-- Mobile menu toggle script -->
<script>
  const btn = document.getElementById('mobile-menu-btn');
  const menu = document.getElementById('mobile-menu');

  btn?.addEventListener('click', () => {
    const isOpen = menu?.classList.contains('hidden') === false;
    menu?.classList.toggle('hidden');
    btn.setAttribute('aria-expanded', String(!isOpen));

    const icon = btn.querySelector('.material-symbols-outlined');
    if (icon) icon.textContent = isOpen ? 'menu' : 'close';
  });
</script>
```
